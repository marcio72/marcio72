# 💼 Projetos em Destaque

## 🛠 Sistema Corporativo de Gestão de Manutenção

Sistema completo desenvolvido em **Java + Spring Boot** para gerenciamento de clientes, máquinas, jogos, solicitações de manutenção e execução de serviços.

### Principais funcionalidades

* Autenticação com Spring Security
* Controle de usuários
* Cadastro de clientes
* Cadastro de máquinas
* Cadastro de jogos
* Solicitações de manutenção
* Execução de serviços
* Dashboard
* Relatórios
* Geração automática de PDFs
* Upload de imagens
* API REST
* Oracle Database

---

## 💬 Serviço de Mensageria com Signal

Um dos grandes diferenciais do sistema é a integração com o **Signal**, permitindo comunicação automática entre o sistema e as equipes responsáveis pelos atendimentos.

O serviço foi desenvolvido em **Java + Spring Boot**, utilizando **Signal-CLI** para envio automático de notificações.

### Funcionalidades

* 📲 Envio automático de novas solicitações
* 🔧 Envio automático das execuções concluídas
* 📋 Resumo detalhado dos serviços realizados
* 👥 Comunicação em grupos do Signal
* ⚡ Notificações em tempo real
* 🔄 Integração total com o sistema principal

Essa solução reduz o trabalho manual e garante que todas as equipes recebam as informações de forma rápida, organizada e confiável.
